# IKEA Clone - Main Page

This is a high-fidelity UI/UX clone of the IKEA India main page, built as a React + TypeScript prototype.

## Features
- **IKEA Brand Design:** Uses the official color palette (`#0058A3`, `#FFDB00`) and Noto Sans typography.
- **Responsive Layout:** Modular grid-based "Showroom" approach that works across devices.
- **Custom Components:**
  - Header with IKEA logo, rounded search bar, and utility icons.
  - Hero section with promotional banner and "Shop now" CTA.
  - Product Grid featuring "Skapa Cards" with hover effects and quick-add cart buttons.
  - Multi-column Footer with customer service and IKEA Family signup sections.

## Tech Stack
- **Framework:** React 19 (TypeScript)
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Modular)

## How to Run
1. Navigate to the `ikea-clone` directory:
   ```bash
   cd ikea-clone
   ```
2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser at `http://localhost:5173`.

## UI/UX Analysis
- **Minimalism:** The design uses generous whitespace to keep the focus on high-quality product imagery.
- **Affordability:** Clean typography and prominent pricing communicate value clearly.
- **Accessibility:** High contrast and large interactive targets ensure a user-friendly experience.
